# ✈ FlyAway Airline Booking Automation Project

## 📌 Project Overview
This project implements a complete QA automation framework for the FlyAway airline booking web application.

## 🛠 Tools & Technologies
- Selenium WebDriver
- TestNG
- Postman
- JSP + Servlet
- Maven
- Jenkins
- Git & GitHub
- Apache Tomcat

## 🔍 Testing Coverage

### 1️⃣ UI Testing (Selenium)
- Flight search
- Flight booking
- Registration
- Payment
- Confirmation

### 2️⃣ Backend Testing (TestNG)
- Unit testing of backend classes
- Assertion-based validation

### 3️⃣ REST API Testing (Postman)

#### Get Flights
GET /api/flights?date=YYYY-MM-DD

#### Get Booked Seats
GET /api/bookedSeats?flightNo=F101

### 4️⃣ Continuous Integration (Jenkins)
- GitHub integration
- Maven build execution
- Automated test execution

## 🚀 How to Run

1. Start Tomcat server
2. Run FlyAway web application
3. Execute Selenium tests via Maven:
   mvn clean test
4. Run Jenkins job for CI execution
